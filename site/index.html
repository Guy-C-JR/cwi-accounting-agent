Skill Signal

Accounting automation built around controls, not blind posting.

The agent turns documents into reviewable expense candidates, stores state in SQLite, preserves workbook formulas, and routes risky records to a Streamlit review cockpit.

The strongest engineering choice is restraint: every high-risk write is backed up, auditable, duplicate-checked, and gated by explicit policy.

Positioning

Automation with accounting-grade caution.

This project supports review hygiene and bookkeeping preparation. It does not claim tax/legal advice or fully autonomous accounting. Its value is in operational reliability: confidence gates, audit trails, backups, and humans where ambiguity matters.
